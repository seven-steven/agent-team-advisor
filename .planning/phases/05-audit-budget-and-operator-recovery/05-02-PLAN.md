<threat_model>

## Trust Boundaries

| Boundary                                       | Description                                                                              |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------- |
| policy JSON -> hook enforcement                | Repo policy values control runtime budget decisions.                                     |
| hook/final-review event -> budget state        | Untrusted hook and final-review inputs select task/session scope and increment counters. |
| budget state -> gate output                    | Local state can cause block/degrade decisions.                                           |
| ordinary advisor artifact -> budget accounting | Recommendation/verdict usage metadata can affect token and latency counters.             |
| final-review artifact -> budget accounting     | Verdict usage metadata can affect token and latency counters.                            |

| --------- | ---------------------- | --------------------------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| T-05-04   | Elevation of Privilege | `scopeKeyForBudget`         | mitigate    | Bind accounting to correlationKey plus taskId/sessionId when available and audit fallback source to reduce bypass by changing one identifier. |
| T-05-05   | Tampering              | `state/advisor-budget.json` | mitigate    | Validate parsed state shape; malformed strict consultation state fails closed with `advisor-budget-state-invalid`.                            |
| T-05-06   | Denial of Service      | budget counters             | mitigate    | Avoid double-counting satisfied retries and re-read ordinary/final-review advisor artifacts; count new consultation/verdict events only.      |
| T-05-SC   | Tampering              | package installs            | accept      | Phase 5 installs no packages; package legitimacy audit lists none.                                                                            |
