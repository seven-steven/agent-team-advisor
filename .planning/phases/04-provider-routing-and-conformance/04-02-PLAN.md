## Trust Boundaries

| Boundary                                       | Description                                                                                                        |
| ---------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| local conformance command -> external gateway  | Local code sends bounded validation requests to an Anthropic-compatible provider using operator-owned credentials. |
| gateway responses -> served-route observation  | Untrusted provider/gateway responses become operator-facing evidence.                                              |
| gateway responses -> conformance artifact      | Untrusted provider responses become local state and audit evidence.                                                |
| conformance command -> `.advisor/` audit/state | Sanitized results are persisted for later audit and operator debugging.                                            |

## STRIDE Threat Register

| Threat ID | Category                | Component                                                                 | Disposition | Mitigation Plan                                                                                                                                           |
| --------- | ----------------------- | ------------------------------------------------------------------------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| T-04-04   | Spoofing / Tampering    | `runSingleConformanceCheck()` live mode                                   | mitigate    | Require separate live evidence for base-message, streaming, tool-use, usage-fields, error-shape, and served-route observation before overall pass.        |
| T-04-05   | Repudiation             | `.advisor/state/provider-conformance.json`, `.advisor/audit/events.jsonl` | mitigate    | Record requested alias, configured route, served-route observation source, per-check status, mode, and timestamp for every run.                           |
| T-04-06   | Information Disclosure  | live conformance request and artifact writer                              | mitigate    | Omit auth headers, bearer tokens, request bodies, prompts, and credential values from errors, artifacts, audit events, and README examples.               |
